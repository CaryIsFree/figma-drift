## Why Phase 0 Matters

**Without Phase 0, you're building in the dark:**

- You: "Let me build this!"
- Reality: Market is already owned by incumbents, no path to profitability, you burn out in 6 months, wasted time

**With Phase 0:**

- You answer 18 hard questions FIRST
- Result: You know if it's worth building before you build it

**Phase 0 Should Filter Out ~80% of Ideas**

Your framework is designed to reject bad ideas early:

- Bad idea #1: "Design tokens monitoring" â†’ Phase 0.1: Market only $5M TAM, incumbents own it â†’ STOP

- Bad idea #4: "Complex enterprise solution" â†’ Phase 0.4: Can't test core assumption quickly â†’ STOP

3. **Seed Tier 3**: Answer questions on Reddit. Don't spamâ€”be genuinely helpful.
